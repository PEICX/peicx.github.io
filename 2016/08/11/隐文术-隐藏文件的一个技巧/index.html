隐文术--隐藏文件的一个技巧

发表于 2016-08-11   |   分类于 生活   |  

介绍：这是一个我新get的小技巧，可以把文本文件txt(不只是文本，其他文件测试也可以）伪装成一个图片jpg文件，随时可以查看原文本内容，但是其他人查看或操作下只是一张普通的jpg图片。

准备：1.你要隐藏或伪装的文本文档。 2. 一张图片（合并后会显示这张图片）

步骤：
windows

  1. 下打开cmd，打开到找到你存放想隐藏文档的文件夹（用dir或cd命令），或着在这个文件夹内新建一个文本内容为start的start.bat 文件，然后打开。
  2. 输入copy /b 图片名称.png + 文本压缩后的压缩包文件.rar 你想生成的图片文件.png（/b的意思以二进制合并）
  3. 回车后就大功告成了，新生成的文件表面上是个图片，但是更改后缀为rar可以显示文本文档的内容。

Linux
1、同一个文件夹下，一张图片，隐藏文件压缩。
2、打开终端，cat xx.rar(压缩文件的的名字) >>xx.png（图片的名字和格式)

如何还原呢？很简单，把文件后缀改为rar就可以查看原来的内容了。
